JavaScript files
================

These JavaScript files are sent to the client's browser and are executed there.

The code uses the following libraries:

- **The Bootstrap framework**: http://getbootstrap.com/
- **d3.js**: http://d3js.org/
- **jQuery**: http://jquery.org/
- **jQuery.cookie**: https://github.com/carhartl/jquery-cookie
- **The gauge class (documented below)**: http://bl.ocks.org/tomerd/1499279
